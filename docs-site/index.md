# @marcohefti/request-network-api-client

TypeScript API client for the Request Network hosted REST API (not the protocol SDK).

- Typed helpers for requests/invoices, payouts, payer/compliance, client IDs, currencies
- Fetch-based HTTP client with retry, logging, and consistent errors
- Subpath exports and tree-shakable domain bundles
